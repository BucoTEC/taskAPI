Tasks API

Btech-solutions

This is a simple REST API built with Express js. It is used for task menagment CRUD operations conected to a Mongo database.

MODELS

Task model

name: { type: String, required: [true, 'must provide name'], trim: true, maxlength: [20, 'name can not be more than 20 characters'], },
completed: { type: Boolean, default: false, },

Routes


/api/tasks

  • GET '/' (get all tasks)
  • POST '/' (create a task)
  • GET '/:id' (get a single task)
  • PATCH '/:id' (update a task)
  • DELETE '/:id' (delete a task)
